# Plan: Core HP x10 + Turret/Builder Integration

## Summary
All 4 cores get x10 HP. Main Core and Core Prime get a custom `FRCoreBlock` class with turret + builder logic. A new stationary flying unit replaces deleted `mainCoreUnit` and `omega` ‚Äî it hovers over the core and provides player-controllable turret/build. The core itself also has independent turret + builder logic.

## Files to Modify/Create

### 1. CREATE: `src/fadingrevelations/content/FRCoreBlock.java`
Custom CoreBlock with turret + builder logic.

**Fields:**
- `float buildRange = 800f` ‚Äî builder radius
- `float buildSpeedMultiplier = 5f` ‚Äî construction speed boost
- `float attackRange = 400f` ‚Äî turret range
- `float reload = 20f` ‚Äî turret reload in ticks
- `BulletType bulletType` ‚Äî default: FlakBulletType with lightning frag

**FRCoreBuild.updateTile():**
1. Call `super.updateTile()` (handles iframes, thrusterTime)
2. Builder: iterate `team.data().plans`, find plans within buildRange, progress `ConstructBuild`
3. Turret: find enemies via `Units.closestTarget()`, rotate toward target, fire when reloaded

**FRCoreBuild.draw():**
1. Call `super.draw()` (core sprite + thrusters)
2. Draw turret sprite overlay (`name + "-turret"`) with rotation toward target
